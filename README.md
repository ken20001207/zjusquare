# ZJU-Square 浙江大学聊天广场

本网站为浙江大学 19级新生林沅霖参加浙大勤创技术推广部秋季纳新的第二阶段面试作品。

## 功能说明

本网站是一个即时通讯网页服务，注册账号后可在本网站与他人讯息交流。

## 前置需求

本网站需要安装与 PHP 网站服务器内使用，
且需要与 MySQL 数据库沟通，以执行其完整功能。
